# Layout Infraestructura de Parques

Este layout está diseñado para mapear la infraestructura de parques en Costa Rica.

## Botones disponibles:

### Infraestructura básica:
- **Poste de luz**: Para mapear puntos de iluminación
- **Edificios**: Estructuras dentro del parque
- **Elementos de parque de niños**: Juegos y áreas infantiles

### Mobiliario urbano:
- **Bancas**: Asientos públicos
- **Mesas**: Mesas para picnic o uso público
- **Basureros**: Contenedores de residuos
- **Bebedero (no funcional)**: Fuentes de agua que no están operativas
- **Puesto de guarda**: Caseta de vigilancia
- **Tipo de piso**: Para indicar el tipo de superficie

### Elementos naturales:
- **Árboles con arbustos**: Vegetación arbórea con arbustos
- **Árbol sin arbusto**: Árboles individuales
- **Plantas**: Otros tipos de vegetación

### Transporte:
- **Parada de bus**: Puntos de transporte público

## Iconos
Todos los botones cuentan con iconos personalizados para facilitar su identificación en el campo.
